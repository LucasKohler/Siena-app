# Dados de exemplo (DEV)

Os arquivos `events.json`, `videos.json`, `users.json` e `attendances.json` nesta pasta são **dados de exemplo para desenvolvimento local**. O `users.json` é uma **allowlist DEV** (telefones fictícios). O `attendances.json` é **mutável em runtime** (presenças gravadas pela API) — adequado apenas a instância única / dev. Não representam calendário oficial, resultados reais nem estatísticas de visualização da associação.

Substituir por fonte oficial (CMS, integração ou banco) quando aprovado em ADR e spec de produção.
